- 🎤 Pitch extraction
- 🎵 Melodic pattern discovery

- 📁 [mirdata](https://github.com/mir-dataset-loaders/mirdata) and [soundata](https://github.com/soundata/soundata) (common loaders for Music and Sound Information Retrival datasets)
